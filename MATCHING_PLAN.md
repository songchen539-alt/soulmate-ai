# SoulMate — 匹配系统MVP方案

## 当前状态 ✅
```
用户 → AI聊天 → Soul Report → 分享
```

## 目标状态 🎯
```
用户 → AI聊天 → Soul Report → 完善资料 → 发现匹配 → 建立连接
```

---

## 第一步：资料系统
用户在拿到Soul Report后，可以完善：

```
昵称/头像
一句话简介
寻找什么样的关系
Soul Type（AI已生成）
五维人格数据（AI已生成）
```

## 第二步：匹配算法
基于Soul Type的五维数据计算兼容度：

```
兼容度 = 100 - |用户A维度 - 用户B维度| 的加权平均值
```

互补维度加分：理性↔感性、冒险↔安稳等

## 第三步：发现页面
```
用户能看到：
- 兼容度排序的潜在匹配
- 对方的Soul Type + 简介
- 想进一步了解 → 发送请求
```

## 第四步：建立连接
```
发送匹配请求 → 对方接受 → 可以聊天
（非AI聊天，而是真人对真人）
```
